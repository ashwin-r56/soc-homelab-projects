# # SOC Homelab Projects

This repository showcases a series of hands-on cybersecurity projects focused on **attack simulation, detection, and defense** using a home lab environment.

These projects demonstrate practical skills relevant to a **SOC (Security Operations Center) Analyst role**, including system hardening, SIEM-based detection, web attack analysis, and privilege escalation defense.

---

## Projects Overview

### 1. Linux Server Hardening

* Implemented SSH hardening and firewall configuration
* Configured Fail2Ban for intrusion prevention
* Simulated brute-force attacks using Hydra
* Demonstrated defense-in-depth strategy

---

### 2. Enterprise SSH Defense & SIEM Detection

* Configured centralized logging using rsyslog
* Monitored logs using a SIEM setup
* Detected brute-force attacks in real time
* Simulated persistence via SSH key injection
* Demonstrated log evasion techniques

---

### 3. Web Server Attack Detection

* Deployed vulnerable web application (DVWA)
* Simulated:

  * Command Injection
  * SQL Injection
  * File Upload (Web Shell)
* Detected attack patterns using HTTP logs
* Analyzed limitations of default logging

---

### 4. Privilege Escalation Detection & Defense

* Simulated multiple privilege escalation techniques:

  * SUID exploitation
  * Cron job abuse
  * Malicious services
  * Binary hijacking
* Implemented mitigation strategies:

  * Permission hardening
  * Service monitoring
  * File integrity monitoring (AIDE)
* Analyzed detection strategies and system protections
